# DCDC_BUCK_ESO_SMC_LabVIEW_Simulink_MATLAB

## 项目简介
本项目构建了一个基于 **滑模控制（SMC）** 与 **扩张状态观测器（ESO）** 的 DC-DC BUCK 变换器控制实验平台。  
系统在 **MATLAB/Simulink** 中完成仿真设计，并通过 **LabVIEW** 平台结合 **NI DAQ** 硬件实现实时控制与数据采集。

## Project Overview
This project develops an experimental control platform for a **DC-DC BUCK converter** based on **Sliding Mode Control (SMC)** and an **Extended State Observer (ESO)**.  
The control algorithms are simulated in **MATLAB/Simulink**, and the real-time implementation is realized in **LabVIEW** with **NI DAQ** hardware for data acquisition and control.
